# 2. Семантические теги HTML5

## Краткое описание

Семантические теги помогают:

* логически разделять страницу;
* делать код более читаемым;
* улучшать SEO;
* помогать браузерам и поисковым системам понимать структуру сайта.

Вместо большого количества `<div>` HTML5 предлагает специальные структурные теги.

---

# Основные семантические теги

## `<header>`

Шапка страницы или раздела.

Обычно содержит:

* логотип;
* название сайта;
* меню навигации.

```html
<header>
    <h1>Мой сайт</h1>
</header>
```

---

## `<main>`

Основное содержимое страницы.

На странице должен быть один `<main>`.

```html
<main>
    Основной контент страницы
</main>
```

---

## `<section>`

Логический раздел страницы.

Например:

* блок новостей;
* описание товара;
* раздел статьи.

```html
<section>
    <h2>Новости</h2>

    <p>
        Последние обновления сайта
    </p>
</section>
```

---

## `<footer>`

Подвал страницы.

Обычно содержит:

* контакты;
* копирайт;
* ссылки;
* дополнительную информацию.

```html
<footer>
    © 2026 Мой сайт
</footer>
```

---

# Структура страницы

## Пример полной структуры

```html
<!-- Начало страницы -->

<header>
    <h1>
        Мой сайт
    </h1>
</header>

<main>

    <section>
        <h2>
            О нас
        </h2>

        <p>
            Мы создаём современные сайты.
        </p>
    </section>

</main>

<footer>
    © 2026 Все права защищены
</footer>

<!-- Конец страницы -->
```

---

# Вложенность тегов

Семантические теги можно вкладывать друг в друга.

Например:

* внутри `<main>` могут находиться несколько `<section>`;
* внутри `<section>` можно размещать заголовки, тексты, изображения и кнопки.

---

## Пример нескольких секций

```html
<!-- Начало страницы -->

<main>

    <section>
        <h2>
            Новости
        </h2>

        <p>
            Последние новости компании.
        </p>
    </section>

    <section>
        <h2>
            Контакты
        </h2>

        <p>
            Email: info@example.com
        </p>
    </section>

</main>

<!-- Конец страницы -->
```

# Семантика + Bootstrap

Семантические теги отлично работают вместе с Bootstrap.

Bootstrap отвечает за:

* внешний вид;
* сетку;
* стили.

HTML5 отвечает за:

* правильную структуру страницы.

---

## Пример с Bootstrap-классами

```html
<!-- Начало страницы -->

<header class="bg-dark text-white p-3">
    <h1>
        Мой сайт
    </h1>
</header>

<main class="container mt-4">

    <section class="mb-4">
        <h2>
            О компании
        </h2>

        <p>
            Мы разрабатываем веб-приложения.
        </p>
    </section>

</main>

<footer class="bg-light p-3">
    © 2026
</footer>

<!-- Конец страницы -->
```

## Задание

Создайте структуру страницы:

* шапка сайта;
* основной контент;
* две секции;
* подвал страницы.

```html
            Первая секция
        </h2>

            Текст первой секции.
        </p>
    </section>

    <section>
        <h2>
            Вторая секция
        </h2>

        <p>
            Текст второй секции.
        </p>
    </section>

</main>

<footer class="bg-light p-3 mt-4">
    © 2026 Учебный сайт
</footer>

<!-- Конец страницы -->
```
