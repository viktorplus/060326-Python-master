Для создания веб-страницы нужные 3 вещи:
* HTML
* CSS
* JS

Первые два пункта — это языки разметки, последний (JS) — полноценный язык программирования.

Для их серьёзного обучения требуется отдельный курс.

Но для создания простой веб-страницы достаточно:
1. базовых знаний HTML 
2. и минимального знания Bootstrap (набор готовых стилей и компонентов)

### Быстрый старт HTML
* [https://www.w3schools.com/html/](https://www.w3schools.com/html/)  

### Быстрый старт Bootstrap

Вместо изучения CSS и JS быстрее и можно кратко познакомиться с Bootstrap
* [https://getbootstrap.com/docs/5.3/getting-started/introduction/](https://getbootstrap.com/docs/5.3/getting-started/introduction/)

(Кстати, с Bootstrap можно познакомиться и в [`w3schools`](https://www.w3schools.com/bootstrap/))


### * CSS

Для простого frontend-решения CSS не нужен.  

ТОЛЬКО если успеете разобрать HTML и Bootstrap и ещё останется время:
* [https://www.w3schools.com/css/](https://www.w3schools.com/css/)
