# 4. Работа с текстом

## Краткое описание

HTML позволяет оформлять текст:

* делать его жирным;
* выделять курсивом;
* создавать списки;
* структурировать информацию.

Такие элементы делают страницу более читаемой и удобной.

---

# Жирный текст

## Тег `<strong>`

Используется для важного текста.

Браузер обычно отображает его жирным.

---

## Пример

```html
<p>
    <strong>
        Важная информация
    </strong>
</p>
```

---

# Курсив

## Тег `<em>`

Используется для смыслового выделения текста.

Обычно отображается курсивом.

---

## Пример

```html
<p>
    <em>
        Выделенный текст
    </em>
</p>
```

---

# Комбинирование тегов

Теги можно вкладывать друг в друга.

---

## Пример

```html
<p>
    <strong>
        Очень важный
    </strong>

    <em>
        текст
    </em>
</p>
```

---

# Маркированные списки

## Теги `<ul>` и `<li>`

`<ul>` — список
`<li>` — элемент списка

---

## Пример

```html
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>Bootstrap</li>
</ul>
```

---

# Нумерованные списки

## Теги `<ol>` и `<li>`

`<ol>` создаёт список с нумерацией.

---

## Пример

```html
<ol>
    <li>Создать файл</li>
    <li>Написать HTML</li>
    <li>Открыть в браузере</li>
</ol>
```

---

# Вложенные списки

Списки можно вкладывать друг в друга.

---

## Пример

```html
<ul>
    <li>
        Frontend
        <ul>
            <li>HTML</li>
            <li>CSS</li>
        </ul>
    </li>
</ul>
```

---

# Bootstrap и текст

Bootstrap предоставляет готовые классы для оформления текста.

---

## Пример

```html
<p class="fw-bold">
    Жирный текст Bootstrap
</p>

<p class="fst-italic">
    Курсив Bootstrap
</p>
```

---

# Bootstrap и списки

Bootstrap умеет изменять стиль списков.

---

## Пример

```html
<ul class="list-group">

    <li class="list-group-item">
        HTML
    </li>

    <li class="list-group-item">
        CSS
    </li>

    <li class="list-group-item">
        Bootstrap
    </li>

</ul>
```

---

# Полный пример

```html
<!-- Начало страницы -->

<main class="container mt-4">

    <h1>
        Изучение веб-разработки
    </h1>

    <p>
        <strong>
            HTML
        </strong>
        отвечает за структуру страницы.
    </p>

    <p>
        <em>
            Bootstrap
        </em>
        помогает быстро оформлять интерфейс.
    </p>

    <h2>
        Что нужно изучить
    </h2>

    <ul class="list-group">

        <li class="list-group-item">
            HTML
        </li>

        <li class="list-group-item">
            CSS
        </li>

        <li class="list-group-item">
            Bootstrap
        </li>

    </ul>

</main>

<!-- Конец страницы -->
```

---

# Мини-практика

## Задание

Создайте страницу:

* с жирным текстом;
* с курсивом;
* с маркированным списком;
* с нумерованным списком;
* с Bootstrap-оформлением списка.

---

## Пример результата

```html
<!-- Начало страницы -->

<main class="container mt-4">

    <h1>
        План обучения
    </h1>

    <p>
        <strong>
            Важно:
        </strong>
        выполнять практику после каждой темы.
    </p>

    <p>
        <em>
            Регулярная практика ускоряет обучение.
        </em>
    </p>

    <h2>
        Технологии
    </h2>

    <ul class="list-group mb-4">

        <li class="list-group-item">
            HTML
        </li>

        <li class="list-group-item">
            CSS
        </li>

        <li class="list-group-item">
            Bootstrap
        </li>

    </ul>

    <h2>
        Этапы обучения
    </h2>

    <ol>
        <li>Изучить HTML</li>
        <li>Освоить CSS</li>
        <li>Изучить Bootstrap</li>
    </ol>

</main>

<!-- Конец страницы -->
```
