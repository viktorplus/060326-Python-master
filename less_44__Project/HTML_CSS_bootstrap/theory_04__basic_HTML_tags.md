# 3. Основные HTML-теги

## Краткое описание

HTML состоит из тегов.

Теги помогают браузеру понимать:

* где находится заголовок;
* где обычный текст;
* где перенос строки;
* где комментарии.

В этом разделе разберём самые базовые теги, которые используются почти на каждой странице.

---

# Заголовки

## Теги `<h1> - <h6>`

HTML содержит 6 уровней заголовков:

* `<h1>` — самый главный заголовок;
* `<h6>` — самый маленький.

---

## Пример

```html
<h1>Главный заголовок</h1>

<h2>Заголовок раздела</h2>

<h3>Подраздел</h3>
```

---

## Важное правило

На странице обычно используется:

* один `<h1>`;
* несколько `<h2>`;
* при необходимости `<h3>`, `<h4>` и далее.

---

# Абзацы

## Тег `<p>`

Используется для обычного текста.

---

## Пример

```html
<p>
    Это обычный абзац текста.
</p>
```

---

## Несколько абзацев

```html
<p>
    Первый абзац.
</p>

<p>
    Второй абзац.
</p>
```

---

# Перенос строки

## Тег `<br>`

Создаёт перенос строки внутри текста.

---

## Пример

```html
<p>
    Первая строка.<br>
    Вторая строка.
</p>
```

---

## Когда использовать `<br>`

Подходит для:

* адресов;
* стихов;
* коротких переносов.

Для больших блоков текста лучше использовать отдельные `<p>`.

---

# Комментарии

## Синтаксис комментариев

Комментарии не отображаются на странице.

Они нужны для:

* заметок;
* пояснений;
* разделения кода.

---

## Пример

```html
<!-- Это комментарий -->
```

---

## Пример в структуре страницы

```html
<!-- Начало блока новостей -->

<section>
    <h2>
        Новости
    </h2>
</section>

<!-- Конец блока новостей -->
```

---

# Пример полноценного блока

```html
<!-- Начало страницы -->

<main class="container mt-4">

    <h1>
        Добро пожаловать!
    </h1>

    <p>
        Это мой первый HTML-документ.
    </p>

    <p>
        HTML помогает создавать структуру сайта.<br>
        Bootstrap помогает оформлять страницу.
    </p>

</main>

<!-- Конец страницы -->
```

---

# Bootstrap и текст

Bootstrap содержит готовые классы для оформления текста.

---

## Пример

```html
<h1 class="text-primary">
    Синий заголовок
</h1>

<p class="text-muted">
    Серый текст
</p>
```

---

# Мини-практика

## Задание

Создайте страницу, содержащую:

* один главный заголовок;
* два подзаголовка;
* несколько абзацев;
* перенос строки;
* комментарии.

---

## Пример результата

```html
<!-- Начало страницы -->

<main class="container mt-4">

    <h1>
        Мой сайт
    </h1>

    <h2>
        О проекте
    </h2>

    <p>
        Это учебный проект.
    </p>

    <h2>
        Контакты
    </h2>

    <p>
        Email: test@example.com<br>
        Телефон: +123456789
    </p>

</main>

<!-- Конец страницы -->
```
