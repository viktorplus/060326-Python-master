Повторим это пример [пример](./theory_03__concatenation_repetition.md#3-неожиданное-поведение-изменяемых-объектов) ещё раз.  

Только на этот раз добавим функцию `id()` для каждого элемента:

```python
n1, n2, n3, n4 = 1, 2, 3, 4
list1 = [n1, n2]
list2 = [n3, n4]

a = [list1, list2]
b = a * 3

print(id(a[0]), id(a[1]))
print(id(b[0]), id(b[1]), id(b[2]), id(b[3]), id(b[4]), id(b[5]))

# 134592101944128 134592101945856
# 134592101944128 134592101945856 134592101944128 134592101945856 134592101944128 134592101945856

b[0][0] = 100
print(a)  # [[100, 2], [3, 4]]
print(b)  # [[100, 2], [3, 4], [100, 2], [3, 4], [100, 2], [3, 4]]

print(id(a[0]), id(a[1]))
print(id(b[0]), id(b[1]), id(b[2]), id(b[3]), id(b[4]), id(b[5]))

# 134592101944128 134592101945856
# 134592101944128 134592101945856 134592101944128 134592101945856 134592101944128 134592101945856
```

Как видим, в обоих списках (и `a`, и `b`) идёт повторений одних и тех же id - `id(list1)` и `id(list2)`
