## Cравнение словарей

### 1. Полное сравнение словарей

#### 1.1. Равенство `==`

```python
d1 = {"a": 1, "b": 2}
d2 = {"b": 2, "a": 1}
d3 = {"b": 2, "a": "1"}

print(d1 == d2)  # True
print(d1 == d3)  # False - разница в типе value: 1 != "1"
```

Как видим, несмотря на ограниченную возможность сортировки, при сравнении словари ведут себя как множества.

---

#### 1.1. Неравенство `!=`

```
print(d1 != d3)  # True
```

### 2. Операции со словарями

С версии 3.9 возможно ТОЛЬКО объединение словарей

```python
d1 = {"a": 1, "b": 2}
d2 = {"b": 3, "c": 5}

d = d1 | d2
print(d)  # {'a': 1, 'b': 3, 'c': 5}
```

Если ключи совпадают, то значение справа заменяет значение слева.

### 3. Сравнение ключей

Ключи (но не словари!) поддерживают все операции множеств

```python"
d1.keys() == d2.keys()

d1.keys() & d2.keys() #

d1.keys() - d2.keys()  # Разность

d2.keys() - d1.keys()  # Разность

d1.keys() ^ d2.keys()  # Симметрическая разность

d1.keys() | d2.keys()  # Объединение
```
