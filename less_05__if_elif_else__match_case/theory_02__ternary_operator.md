В Python тернарный оператор — это способ записать условную конструкцию в одну строку.  
Подходит ТОЛЬКО для простых конструкций.  
Но даже в этом случае проигрывает классическому блочному оператору в читабельности.

Синтаксис:

```python
<значение_если_True> if <условие> else <значение_если_False>
```

### Пример 1: простое условие

```python
x = 10
result = "Положительное" if x > 0 else "Отрицательное"
print(result)  # Положительное
```

Эквивалентно:

```python
if x > 0:
    result = "Положительное"
else:
    result = "Отрицательное"
```

---

### Пример 2: вложенный тернарный оператор

```python
x = 0
result = "Положительное" if x > 0 else "Отрицательное" if x < 0 else "Ноль"
print(result)  # Ноль
```

Используется КРАЙНЕ редко, т.к. плохо читается.

---

### Пример 3: с функциями

```python
x = 0
print("Положительное") if x > 0 else print("Отрицательное") if x < 0 else print("Ноль")
```

Или ещё короче: весь тернарный оператор использовать в качестве аргумента функции.

```python
x = 5
print("Положительное" if x > 0 else "Отрицательное" if x < 0 else "Ноль")
```
---

### Пример 4: с числами

```python
a, b = 5, 10
max_val = a if a > b else b
print(max_val)  # 10
```


### * Правило преобразования тернарного оператора в блочный и наоборот

Обычно длинный тернарный оператор в коде - **моветон**.

Поэтому сами стараемся избегать таких нечитабельных выражений.

Однако, если в чужом коде попадётся длинный нечитаемый тернарный оператора, поступаем так:

* "разбираем" его на блоки слева направо
* то, что стоит перед `if` - ставим внутрь блока `if`
* `if`, который стоит после`else`, - превращаем в `elif`

```python
x = 5
print("Положительное") if x > 0 else print("Отрицательное") if x < 0 else print("Ноль")

if x > 0:
    print("Положительное") 
elif x < 0:
    print("Отрицательное") 
else:
    print("Ноль")
```

### ** Более сложный вариант

```python
x = 5
print("Большое положительное") if x > 100 else print("Положительное") if x > 0 else print("Ноль") if x == 0 else print("Отрицательное") if x > -100 else print("Большое отрицательное")


if x > 100:
    print("Большое положительное")
elif x > 0:
    print("Положительное")
elif x == 0:
    print("Ноль")
elif x > -100:
    print("Отрицательное")
else:
    print("Большое отрицательное")
```
