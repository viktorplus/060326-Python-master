## Функция `open()` 

Используется для открытия файлов и работы с ними: 
* чтения, 
* записи, 
* добавления данных

### Синтаксис

```python
open(filename, mode='r', encoding=None)
```

### Параметры

* `filename` — путь к файлу.
* `mode` — режим открытия файла.
* `encoding` — кодировка файла (обычно `utf-8` для текстовых файлов).

### Основные режимы открытия

| Режим | Описание                                               |
| ----- | ------------------------------------------------------ |
| `r`   | Чтение файла (по умолчанию)                            |
| `w`   | Запись (создаёт новый файл или очищает существующий)   |
| `a`   | Добавление данных в конец файла                        |
| `x`   | Создание нового файла, ошибка если файл уже существует |
| `b`   | Бинарный режим                                         |
| `t`   | Текстовый режим (по умолчанию)                         |
| `+`   | Чтение и запись одновременно                           |

### Пример чтения файла

```python
file = open("data.txt", "r", encoding="utf-8")
text = file.read()
file.close()  # ⚠️ Открытый файл ОБЯЗАТЕЛЬНО должен быть закрыт

print(text)
```

### Пример записи в файл

```python
file = open("data.txt", "w", encoding="utf-8")
file.write("Привет, мир!")
file.close()
```

### Контекстный менеджер

Конструкция `with` автоматически закрывает файл после завершения работы.

```python
with open("data.txt", "r", encoding="utf-8") as file:
    text = file.read()

print(text)
```
